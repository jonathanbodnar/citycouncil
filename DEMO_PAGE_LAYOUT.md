## Responsive Behavior

### Mobile (< 640px)
- 2-column grid for talent cards
- Stacked layouts
- Touch-optimized hit areas
- Full-screen video

### Tablet (640px - 1024px)
- 3-column grid for talent cards
- More spacing
- Larger touch targets

### Desktop (> 1024px)
- 4-column grid for talent cards
- Mouse wheel support
- Hover states
- Larger UI elements

## Color Coding

- **Blue**: Primary actions (CTA, links)
- **Red**: Like/heart, branding accents
- **White**: Text, icons (high contrast)
- **Glass**: UI overlays (frosted glass effect)
- **Gradient**: Backgrounds (dark themed)
